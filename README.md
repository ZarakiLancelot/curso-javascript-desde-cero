## 🧑‍💻 Cómo usar este repositorio

1. Clona este repo en tu computadora:

   ```bash
   git clone https://github.com/tuusuario/curso-javascript-desde-cero.git
   ```

2. Abre la carpeta en Visual Studio Code.
3. Instala la extensión **Live Server** (opcional, pero recomendado)
4. Haz clic derecho sobre `index.html` -> _Open with Live Server_
5. Abre la consola del navegador (`F12`) y sigue las instrucciones de cada lección.

## 🎯 Objetivo del curso

- Entender **dónde corre JavaScript** y cómo funciona en el navegador y con Node.js
- Conocer y usar las **DevTools** para depuración.
- Configurar un entorno básico con **HTML**, **CSS** y **JS**
- Practicar con la consola, errores comunes y breakpoints.
- Realizar **interacciones simples con el DOM**
- Resolver **ejercicios prácticos** paso a paso.
